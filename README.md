# Thread_Intelligence_Feed_Reader
Threat Intelligence Feed Reader:  A modern, responsive React application for monitoring and analyzing threat intelligence feeds. This dashboard provides real-time visualization of Indicators of Compromise (IOCs) including IP addresses, domains, file hashes, and URLs with severity filtering and search capabilities. 
🚀 Features
Live Dashboard: Real-time statistics on threat severity levels (Critical, High, Medium, Low)
Multi-format Support: Monitors IPs, Domains, Hashes, and URLs
Advanced Filtering: Search by indicator value or description, filter by severity level
One-click Copy: Quickly copy IOCs to clipboard for integration with security tools
Responsive Design: Fully responsive layout optimized for desktop and mobile viewing
Visual Severity Indicators: Color-coded badges for quick threat assessment
Auto-refresh: Simulated live feed updates with manual refresh capability
🛠️ Tech Stack
Frontend Framework: React 18
Build Tool: Vite
Styling: Tailwind CSS
Icons: Lucide React
Language: JavaScript (ES6+)
📦 Installation
Prerequisites
Node.js 16.0 or higher
npm or yarn package manager
